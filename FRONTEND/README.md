# Frontend - Final (Create React App + Tailwind + Lucide Icons)

## Quick Start
1. Install dependencies:
   cd frontend_final
   npm install

2. Run:
   npm start

3. Ensure backend API is running at http://localhost:5000

## Changes
- Full responsiveness across pages.
- Icon-based actions (Lucide React).
- Dark theme fixed for card contrast.
- Modern tables with alternate styles and hover.
- Orders page: revenue summary + full orders table.
- Dashboard: shows 5 most recent orders.
